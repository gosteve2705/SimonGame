Simon Game Rules

  1. press any key to Start.
  2. Simon will give the first colour. Repeat the Colour by pressing the same color Button.
  3. Simon will duplicate the first colour and add one. Repeat these two colours by pressing the same color buttons, in the same order.
  4. Simon will duplicate these first two colours and add one.
  5. Continue playing as long as you can repeat each sequence of colours correctly.
  6. After the 5th, 9th and 13th colours in a sequence, Simon automatically speeds up.
  7. If you fail to repeat a sequence exactly, or if you take more than 5 seconds to repeat a signal, Simon responds with a "RAZZ" sound and a Red Background. This means you've lost, and the sequence of colours ends.
